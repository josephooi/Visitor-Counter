counter.txt
Contains a single number that is the number of people who have visited the website

display.cgi
Python script that displays on the webpage the number of people who have visited the website

count.cgi
Python script that increments the number in counter.txt by 1

index.php
Webpage that keeps track of the number of people who have visited the webpage thus far and displays the browser that the current user is using.
